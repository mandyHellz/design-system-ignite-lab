# Design System

Project created at Rocketseat's Ignite Lab event, where we built a complete Design System using technologies and tools such:

- Figma
- React with Vite
- Typescript
- Tailwind
- Radix Ui
- Storybook
- CD/CI with GitHub actions
- Jest and Testing Library
